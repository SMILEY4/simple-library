- overlay component with known id that functions as anchor/plane for all dialogs
    - invisible
    - fills parent component
    - renders dialogs via provided state

- hook to manage dialogs
    - takes functions to access some state (local/global/fixed, ...) as params
    - adds/removes dialog-entries from state
    - callback functions for when user closes dialog
    - only access "internal state" of dialog after user "closes" dialog
        - close "event" = reason + payload
    - every dialog has unique id = id of div
        - use for anchor of dropdowns, contextmenus, ...
    - takes reactElement as child or function "(dialogId) => reactElement"
    - hook for opening generic or predefined dialogs (config, warn, error, ...)
